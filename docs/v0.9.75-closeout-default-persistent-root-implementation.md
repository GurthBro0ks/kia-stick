# KIA-Stick-v0.9.75-closeout-default-persistent-root-implementation

v0.9.75 updates `scripts/closeout-helper.mjs` so default proof discovery prefers the persistent KIA proof root when available.

Behavior:

- `--proof-dir` remains the strongest selector and reviews exactly the supplied proof directory.
- `--proof-root` remains an explicit root selector and is labeled as `default_latest_from_explicit_proof_root`.
- `KIA_PROOF_ROOT` is honored only when it resolves inside an allowed proof root.
- `/home/mint/kia-stick-local-proofs` is selected by default when it exists and contains `proof_kia_stick_...` directories.
- `/tmp` is used only as fallback.

The helper output includes `proof_discovery_mode`, `proof_discovery_note`, `proof_root`, and `proof_dir` so operators can see what was inspected.

The helper remains read-only. It does not create, delete, rewrite, or mutate proof directories. It does not execute `git push`.

Tests cover explicit root labeling, safe `KIA_PROOF_ROOT`, unsafe `KIA_PROOF_ROOT` rejection, persistent default selection, and unchanged explicit `--proof-dir` behavior.
