# KIA-Stick-v0.9.76-operator-closeout-proof-root-guidance

Operator guidance for closeout proof roots:

- Use `npm run closeout:summary -- --proof-dir <proof_dir>` for the active bundle closeout.
- Use `npm run closeout:review -- --proof-dir <proof_dir>` for the active bundle review.
- Default `npm run closeout:summary` and `npm run closeout:review` should choose `/home/mint/kia-stick-local-proofs` when persistent KIA proofs exist.
- `/tmp` is fallback only.
- Stale old `/tmp` proofs should not drive current closeout decisions when persistent proofs exist.
- Stop-on-WARN/FAIL behavior is preserved.
- Explicit `--proof-dir` remains the safest operator path for active bundle gates.
