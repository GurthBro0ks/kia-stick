`scripts/fake-browser-qa-evidence.mjs` reports:

- proof PASS/WARN/FAIL review field.
- manual QA status.

- real-doc and browser file capability blocked status.

## Closeout Meaning

Manual QA is `PASS` after `OPERATOR_QA_PASS for /home/mint/kia-stick-local-proofs/proof_kia_stick_v0_9_18_to_v0_9_22_fake_only_qa_evidence_proof_readiness_bundle_20260628T111708Z`.
